### Common Issues

**"Python not found"**
```
Solution: Install Python 3.8+ from python.org

```

**"FFmpeg not found"**
```
Solution: Install FFmpeg

```

```
3. Check file isn't corrupted

```

```
1. Reduce EQ bands to 16
2. Disable glow effects
3. Lower zoom to 50%
4. Close other apps
```
